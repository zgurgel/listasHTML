Passos para clonar um repositório

  1. Cria um repositório no github
  2. Copia a URL do repositório
  3. Abre o terminal
  4. Dá o comando: git clone "https://..."
  5. Muda de pasta: cd nome_do_repositorio
  6. Dá o comando para entrar no vscode: code .

Após atualizar o código

  1. git add .
  2. git commit -m "descricao"
  3. git push origin main
